# Node.js + Stripe API Checkout Integration

A clean, production-ready boilerplate for handling secure e-commerce payments.

## Features

- **Server-side Session Creation:** Prevents price tampering.
- **Stripe Webhooks:** Securely confirms payments using signature verification.
- **Environment Variables:** Keeps sensitive API keys out of the codebase.
- **Mobile Responsive:** Uses Stripe's hosted checkout for 100% responsiveness.

## Tech Stack

- Node.js / Express
- Stripe SDK
- Dotenv for Security
